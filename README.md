# PetPet - 桌宠

一个简单的 Python 桌宠程序，具有透明背景和隐藏窗口装饰的特性。

## 功能特性

- 🎨 **透明背景** - 背景完全透明，融合于桌面
- 🪟 **隐藏窗口装饰** - 隐藏标题栏和窗口边框
- 🎭 **情感系统** - 支持不同的表情（正常、高兴、伤心）
- 🖱️ **可拖拽** - 用鼠标左键拖拽宠物移动，宠物会做出高兴的表情
- 🎮 **自动活动** - 宠物会自动在窗口内移动
- 🔝 **窗口置顶** - 始终显示在其他窗口上方

## 安装

1. 确保已安装 Python 3.6+
2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 使用

运行程序：
```bash
python main.py
```

## 操作说明

- **左键拖拽** - 拖拽宠物移动（宠物会做出高兴的表情）
- **右键点击** - 关闭程序

## 技术栈

- tkinter - GUI 框架
- Pillow(PIL) - 图像处理
- Python 标准库

## 自定义

你可以修改 `create_pet_image()` 方法来改变宠物的外观，或者调整以下参数：

- `self.width` 和 `self.height` - 窗口大小
- `self.vx` 和 `self.vy` - 移动速度
- 颜色值 - 修改宠物和表情的颜色
